# 文件重命名工具 - 打包说明

## 📋 概述

这是一个基于Python tkinter的文件重命名工具，可以批量重命名包含特定文本的文件。该工具提供了图形界面，使用简单直观。

## 🚀 快速开始

### 方法一：使用批处理文件（推荐）

1. **双击运行 `快速打包.bat`**
   - 自动检查Python环境
   - 自动安装PyInstaller
   - 自动打包生成可执行文件

2. **等待打包完成**
   - 打包过程可能需要2-5分钟
   - 生成的文件位于 `dist/文件重命名工具.exe`

### 方法二：手动打包

```bash
# 1. 安装PyInstaller
pip install pyinstaller

# 2. 打包命令
pyinstaller --onefile --windowed --name="文件重命名工具" --clean "优化版文件重命名UI.py"
```

## 📁 文件说明

| 文件名 | 说明 |
|--------|------|
| `优化版文件重命名UI.py` | 优化版源代码（推荐打包） |

## 🔧 使用方法

### 1. 启动程序
- 双击 `文件重命名工具.exe`

### 2. 选择文件夹
- 点击"浏览文件夹"按钮
- 选择包含要重命名文件的文件夹

### 3. 设置替换规则
- **查找文本**: 输入要查找的文本
- **替换为**: 输入要替换成的文本

### 4. 预览更改
- 点击"预览更改"查看将要进行的操作
- 确认无误后再执行

### 5. 执行重命名
- 点击"执行重命名"
- 确认操作后开始重命名

## ⚠️ 注意事项

1. **备份重要文件**: 重命名操作不可撤销，建议先备份
2. **预览确认**: 执行前务必使用预览功能确认
3. **文件占用**: 确保要重命名的文件没有被其他程序占用
4. **权限问题**: 确保对目标文件夹有写入权限

## 🐛 常见问题

### Q: 打包失败怎么办？
A: 
1. 确保Python环境正确安装
2. 尝试升级PyInstaller: `pip install --upgrade pyinstaller`
3. 检查是否有杀毒软件干扰

### Q: 可执行文件很大怎么办？
A: 
1. 使用优化版源代码打包
2. 确保排除了不必要的模块
3. 可以使用UPX压缩（可选）

### Q: 程序启动慢怎么办？
A: 
1. 首次运行需要解压，属于正常现象
2. 后续运行会快很多
3. 可以考虑使用--onedir模式打包

### Q: 在其他电脑上无法运行？
A: 
1. 确保目标电脑是Windows系统
2. 检查是否被杀毒软件拦截
3. 尝试以管理员身份运行
